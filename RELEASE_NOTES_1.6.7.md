# Release 1.6.7

## Changes

### Fixed
- Fixed `__version__` constant to correctly report 1.6.7 (was incorrectly 1.6.5 in 1.6.6 release)

## Installation

```bash
# Via uv (recommended)
uv tool install rubber-ducky

# Via pip
pip install rubber-ducky

# Upgrade
uv tool upgrade rubber-ducky
pip install --upgrade rubber-ducky
```
